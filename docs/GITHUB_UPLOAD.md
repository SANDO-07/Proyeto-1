# Cómo subir a GitHub

## Opción 1: GitHub Desktop
1. Crea un repositorio nuevo en GitHub.
2. Copia esta carpeta completa en tu computadora.
3. Abre GitHub Desktop.
4. Add local repository.
5. Publish repository.
6. En GitHub ve a **Settings > Collaborators**.
7. Agrega a: `rotizs@miumg.edu.gt`

## Opción 2: Git en terminal
```bash
git init
git add .
git commit -m "Entrega Proyecto 1 - MVP incendios forestales"
git branch -M main
git remote add origin TU_URL_DEL_REPOSITORIO
git push -u origin main
```
